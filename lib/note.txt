These libraries are mainly compiled Java classes assembled into JARs
that support the CWF code.  The main subdirectories are for native
libraries for each supported operating system.  The JAR files are as
follows:

Main CWF JAR (main code, GUI help pages, other resource files):
  cwf.jar

Alloy Look and Feel (used for Unix GUI only):
  alloy.jar

GeoTools (shapefile support):
  geoapi-1.1.0alpha.jar
  gt2-main-nojai.jar
  gt2-shapefile.jar
  JTS-1.4.jar

GIF encoding:
  neuquant.jar (neural net quantization)
  gifencoder.jar (actual format encoder)

PDF encoding:
  itext-1.2.jar

XML parsing:
  jaxen-full.jar
  saxpath.jar

C-style print formatting:
  hb16.zip

Command line argument parsing:
  jargs.jar

Java Advanced Imaging CODEC (mainly for GeoTIFF encoding):
  jai_codec.jar

Mathematical expression parsing:
  jep-2.24.jar

Plotting symbols:
  PlotPackage.jar

Java matrix manipulation:
  Jama-1.0.1.jar

HDF native stubs:
  jhdf.jar

Unidata UDUNITS support:
  udunits.jar

NetCDF Java support:
  bzip2.jar
  commons-httpclient-2.0.1.jar
  commons-logging.jar
  dods-1.1.7.jar
  gnu-regexp-1.1.4.jar
  grib.jar
  jdom-b9.jar
  jpeg2000.jar
  netcdf-2.2.jar
  visad.jar
