  \item[IDL] Interactive Data Language

  \item[RGB] Red/Green/Blue

  \item[SDS] Scientific Data Set

  \item[SST] Sea Surface Temperature

  \item[TDF] TeraScan Data Format

  \item[THG] The HDF Group

  \item[UTC] Coordinated Universal Time

  \item[UTM] Universal Transverse Mercator

  \item[WGS] World Geodetic System

  \item[XML] Extensible Markup Language
