CoastWatch Data Analysis Tool Help: Using the data view controls

You can use the data view controls to change the data displayed using a point-and-click interface on the data view:

Fit image to window
Fits the entire extent of the data file in the view.
Zoom in
Zooms in on the data view so that a smaller area is shown. Double-clicking the left mouse button on the view also zooms in.
Zoom out
Zooms out on the data view so that a larger area is shown. Double-clicking the right mouse button on the view also zooms out.
Zoom to selection
Sets the view to show an area of interest that you select.
Zoom to actual size
Sets the view magnification so that each data pixel is exactly one screen pixel in size.
Fill window
Fills the data view window and eliminates any black no-data edges. Some of the data pixels may be cropped out.
