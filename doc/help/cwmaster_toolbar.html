CoastWatch Master Tool Help: Using the toolbar

You can use the toolbar buttons to change projection and region parameters by clicking and dragging on the master view. Each tool modifies the master parameters in a different way:

Magnify
Decreases the region pixel size by a factor of 2 in both width and height. This has the effect of defining a smaller earth area in the master.
Shrink
Increases the region pixel size by a factor of 2 in both width and height. This has the effect of defining a larger earth area in the master.
Zoom
Allows you to select a rectangular area on the master view. The rectangle is used to derive a new region center location and pixel size.
Pan
Allows you to click and drag the master view. The final position is used to derive a new region center location.
Recenter
Allows you to select a new region center location.
Reset
Resets all projection and region parameters to the default values: a Mercator projection centered on the US. Use this to get back to a sane set of master parameters.
