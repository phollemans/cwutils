EDAC Help: Overview

The CoastWatch Earth Data Access Client (EDAC) enables CoastWatch data users to search through a catalog of available remote sensing data and access data of interest. Data may be viewed interactively using the View tab or searched using the Catalog tab. The data view allows users to zoom in and out, change the color palette and scaling, overlay coast, grid, and political lines, and show the latitude/longitude coordinates and data value at a location on the map. The catalog search allows users to search by region coverage and date/time and display either thumbnail images or detailed search results.
