CoastWatch Data Analysis Tool Help: Using the data view toolbar

You can use the toolbar buttons to change the data displayed using a point-and-click interface on the data view. Each tool modifies the data view in a different way:

 Magnify
Magnifies the data view features by two times so that a smaller area of the earth is shown in the view.
 Shrink
Shrinks the data view features by two times so that a larger area of the earth is shown in the view.
 Actual
Sets the view magnification so that each data pixel is exactly one screen pixel in size.
 Zoom
Allows you to select a rectangular area of interest on the data view. The view is magnified to encompass the new zoom rectangle.
 Pan
Allows you to click and drag the data view to a new area without changing the magnification.
 Recenter
Allows you to select a new data view center location, similar to panning but with a single click.
 Fit
Resets the data view magnification and eliminates any black edges, so that the full extent of the view is used. Some of the data may be cropped at the edges in order to accomplish the fit.
 Reset
Resets the data view magnification so that the entire extent of the data is shown.
